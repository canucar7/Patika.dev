Request:

[7, 5, 1, 8, 3, 6, 0, 9, 4, 2]
Write the steps of Binary-Search-Tree in given array.

Answer:
First number is selected as a root. If the next number bigger, place it to right ,if not place it to left. Keep continue until the end.
        7
      /   \
     5     8
    / \     \
   1   6     9
  / \
 0   3
    / \
   2   4
